## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- NodeJS - [v16.x](https://nodejs.org/en/)
- NPM

### Setup

```bash
git clone https://github.com/mmatongo/hello-rails-front-end.git
cd hello-rails-front-end
```

### Install

```bash
yarn
```

### Usage

```bash
npm run start
```
